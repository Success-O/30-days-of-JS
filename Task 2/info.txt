What is the difference between null and undefined?
In JavaScript, undefined is a type, whereas null an object. It means a variable declared, but no value has been assigned a value. Whereas, null in JavaScript is an assignment value. You can assign it to a variable.

What is NaN in JavaScript? What is the type of NaN?

NaN stands for Not-a-Number. It is a special value of the NaN property of the global object. NaN indicates a failed operation on numbers. Use isNaN() function to check if a value is NaN .The type of NaN , which stands for Not a Number is, surprisingly, a number. ... However, it is a numeric data type whose value cannot be represented using actual numbers. So, the name "Not a Number", doesn't mean that the value is not numeric.
